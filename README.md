# My Espanso Matches

This is a collection of my [Espanso](https://espanso.org/) matches.

## What is Espanso?

Espanso is a cross-platform text expander for Windows, macOS, and Linux. It allows you to expand abbreviations into longer text.

## How to use these matches?

1. Install Espanso: [https://espanso.org/install/](https://espanso.org/install/) and follow the [Get Started guide](https://espanso.org/docs/get-started/).
2. Copy the matches you want to use to the espanso config directory. You can find the espanso config directory by running: `espanso path`. The matches are stored in the `match` directory.
3. Some matches use environment variables. You can copy the `.env.example` file to your config directory and add the variables you need.
